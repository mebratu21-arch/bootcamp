# Basic React Router Example

This is the simplest possible React Router setup.

## How to Use

To test this basic example, update `src/main.jsx`:

```jsx
import App from './day-4/basic/App.jsx'
```

## What It Demonstrates

- `BrowserRouter` wraps the entire app
- `Link` components for navigation (instead of `<a>` tags)
- `Routes` contains all route definitions
- `Route` defines path and element to render

## Exercise Ideas

1. Add a new page (e.g., Services)
2. Add inline styles to make it look better
3. Try navigating and watch the URL change
