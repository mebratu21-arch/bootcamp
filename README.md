## Quick Start

### Prerequisites

```bash
node --version  # v22.x or higher
npm --version   # v10.x or higher
python --version  # 3.12+ (for AI/Prompt Engineering projects)
```

### Installation

```bash
# Clone repository
git clone https://github.com/mebratu21-arch/bootcamp.git
cd bootcamp

# Run any React/Vite project
cd age-tracker  # or any project directory
npm install
npm run dev

# Run backend API
cd "task managment api"
npm install
npm start
```

### Running Multiple Projects

```bash
# Terminal 1 - Frontend
cd search-book-app && npm run dev -- --port 5173

# Terminal 2 - Backend API
cd "task managment api" && npm start

# Terminal 3 - Another Frontend
cd redux-thunk-users && npm run dev -- --port 5174
```
