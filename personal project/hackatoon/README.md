Global Country Explorer

Hackathon Project – MVP / Full App**  

A modern, interactive web application that allows users to explore countries worldwide in real-time, with search, filter, sort, and detailed information. Built with **HTML, CSS, JS**, and the **REST Countries API**.


 Features

- **Real-time data:** Fetches all countries using the REST Countries API.
- **Search:** Quickly find countries by name.
- **Filter by region:** Africa, Asia, Europe, Americas, Oceania.
- **Sort:** Alphabetically (A→Z, Z→A) or by population (ascending/descending).
- **Clickable country cards:** Open a modal with detailed info:
  - Flag  
  - Capital  
  - Subregion  
  - Population  
  - Languages
- **Dark/Light mode toggle** with persistent preference using LocalStorage.
- **Responsive UI**: Works on desktop and mobile.
- **Persistent user preferences**: Search text, filter, sort, and theme saved in LocalStorage.

---

## Tech Stack

- HTML5  
- CSS3 (with responsive design & dark/light mode)  
- Vanilla JavaScript (ES6+)  
- REST Countries API ([https://restcountries.com/](https://restcountries.com/))  
- LocalStorage for saving preferences
