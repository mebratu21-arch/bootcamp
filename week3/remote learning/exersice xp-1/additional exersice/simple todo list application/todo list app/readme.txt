# Todo List App

A simple **Todo List application** built with HTML, CSS, and JavaScript. Users can:

- Add new tasks
- Mark tasks as complete
- Remove individual tasks
- Remove all tasks
- Filter tasks by status (All, Active, Completed)

---

## Features

- **Add Task:** Type a task and click "Submit" to add it to the list.
- **Mark Complete:** Click on a task to mark it as completed.
- **Delete Task:** Remove individual tasks from the list.
- **Delete All Tasks:** Clear all tasks with a single button.
- **Filter Tasks:** Show all tasks, only active tasks, or only completed tasks.
- **Material Design Styling:** Clean and modern interface.
- **Button State Management:** "Remove All" button disabled when there are no tasks.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
