About

The Color Guessing Game is built using HTML, CSS, and JavaScript.
Users are presented with a random RGB color value and must select or input the correct color. The game supports three difficulty levels: Easy, Medium, and Hard.

Itâ€™s perfect for learning RGB color values in a fun and engaging way.

Features

Random RGB color generation

Three difficulty levels:

Easy: 3 color options

Medium: 6 color options

Hard: Input the exact RGB value

Instant feedback on guesses (Correct/Wrong)

5-second countdown before generating the next color after a correct guess

Responsive and user-friendly design

Installation

Follow these steps to run the game locally:

# Clone the repository
git clone https://github.com/your-username/color-guessing-game.git

# Navigate into project directory
cd color-guessing-game

# Open index.html in your browser


No additional setup or dependencies are required.

Usage

Open index.html in your browser.

Choose your difficulty level: Easy, Medium, or Hard.

Guess the color based on the RGB value displayed.

Receive instant feedback.

If correct, a 5-second countdown will show the next color.
